# LLM Bug Detection Research Agent

An AI-powered tool built during undergraduate research at Virginia Tech to automate the review of research papers on LLM bug detection.

## What it does
- Summarizes and filters 100+ research papers using LLMs
- Classifies articles and prioritizes high-value research
- Reduces manual review time by ~50%

## Tech Stack
- Python
- LLM APIs (e.g. OpenAI/Anthropic)
- Automated document processing pipelines

## Context
Built as part of undergraduate AI research at Virginia Tech (May–August 2025).
